# Property Schema

## System-generated fields
- apartment_id: integer, primary key, auto-generated by database

## Optional metadata
- URL: optional, can be null for manually added properties

## Required user/backend fields
- Listing_type: string, required, allowed values: sale, rent
- Location: string, required
- Bedrooms: integer, required
- Bathrooms: integer, required
- Area_sqm: float, required
- Floor: string, required
- Floor_type: string, required
- Description: text, required
- Specialities: text, required

## Price rules
- Sale_price: float, required only if Listing_type = sale
- Price_annualy: float, required only if Listing_type = rent
- Price_monthly: optional, not required for the current recommender pipeline

## Boolean fields
- Furnished: boolean, required
- Pool: boolean, required
